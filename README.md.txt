# Automated KB Testing

## Overview
Automated testing framework for RAGAS (Retrieval Augmented Generation Assessment) applications.

## Features
- Knowledge base retrieval testing
- Generation quality assessment
- Automated evaluation metrics
- Performance benchmarking

## Instructions to run the application
```bash
pip install -r requirements.txt
streamlit run streamlit_ragas_eval.py
```

## Test Categories
- **Retrieval Tests**: Accuracy of document retrieval
- **Generation Tests**: Quality of generated responses
- **End-to-End Tests**: Complete pipeline validation
